Instala Tequio como app

Acceso directo desde la pantalla de inicio de tu teléfono o computadora. Sin abrir el navegador, sin escribir la URL. Funciona también sin internet.

¿Por qué instalar?

  • Ícono en home screen — abre Tequio con un toque, como cualquier app
  • Sin barra de URL — pantalla completa para mejor experiencia
  • Funciona offline — leyes y datos cacheados para consultar sin internet
  • Cero descarga — usa el mismo navegador, no ocupa espacio extra
  • Actualizaciones automáticas — siempre tendrás la última versión
🤖
Android / Chrome
Pasos para instalar
  1. Toca el botón "📲 Instalar ahora" arriba
  2. Si no aparece el modal, abre el menú del navegador (⋮)
  3. Selecciona "Instalar app" o "Añadir a pantalla de inicio"
  4. Confirma y Tequio aparecerá como app en tu pantalla
🍎
iPhone / Safari
Pasos para instalar
  1. Toca el botón Compartir ⬆️ en la barra inferior
  2. Desplaza hacia abajo y selecciona "Añadir a pantalla de inicio"
  3. Toca "Añadir" en la esquina superior derecha
  4. Tequio aparecerá como app en tu home
💻
Desktop / Chrome o Edge
Pasos para instalar
  1. Toca "📲 Instalar ahora" arriba
  2. O busca el ícono de instalación en la barra de URL (🖥️+)
  3. O abre el menú del navegador y selecciona "Instalar Tequio..."
  4. Tequio se abrirá como ventana independiente
🦎 Tequio es una PWA (Progressive Web App) — la misma web pero con superpoderes de app.
Sin descargas desde Play Store / App Store. Cero costo. Cero anuncios.
